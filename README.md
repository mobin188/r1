# r1

A Flask Python web application.

## Local Development

```bash
# 1. Clone the repo
git clone https://github.com/mobin188/r1.git
cd r1

# 2. Install dependencies
pip install -r requirements.txt

# 3. Copy environment file
cp .env.example .env
# Edit .env with your SECRET_KEY etc.

# 4. Run the app
python run.py
